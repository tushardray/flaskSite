How to interact with the table:


  1. Create a new student using the "Create new student" link just above the table.
  2. Edit an existing student using the "Update student" link on the right end of the table
  3. Delete a student by clicking the "Delete student" button on the rightmost side of the table




How it works:

  • After creating a student, a line containing the new information is added to a CSV file containing all other student information. This information is then added to the table.
  • When deleting a student from the table, the program searches the file for the ID number matching the one selected by the user. The particular line is removed, resulting in removal from the table.
  • The updating process is very similar to that used to delete names. The only difference is that instead of deleting all of the student's information, only the parts that were changed in the update are removed. They are replaced with the new information and the table updates.
